The Things You'll Need:

1) Domain Name

Don’t have a domain name yet? Get one at hover.com. Hover is a SUPER user-friendly registrar with a good support team. They also don’t upsell you on a bunch of stuff like GoDaddy does.

2) Hosting

There are a lot of options out there. I’d recommend bluehost, host gator, or dreamhost. These should run you $5-$9 / month. You could also host your site for free on GitHub pages. The default URL would be "username.github.io." You can set up your own URL by changing the DNS settings on hover. There is some technical setup required here which I’ll get to later.

3) FTP Client

If you don't use GitHub to host, you'll have to use an FTP client. An FTP (File Transfer Protocol) client is software that helps you put files on the web. It’s kinda weird when you first think about it, but websites are really just folders of information exactly like the ones on your hardrive. I recommend CyberDuck. Once you get login credentials from your hosting company, then you can upload the folder full of code/images/etc. that makes up your site. Then you’ll have a website on the world wide web! Magic!
